# MetaVirus_Utilities
MetaVirus相关工具集合

## 战斗测试工具
战斗测试工具用于在本地模拟一个战斗环境，使用专用的战斗测试客户端进行战斗相关的调试及测试

- **战斗测试工具必须配合 数据项目 ([MetaVirus_GameData](https://github.com/Metavirus-Game/MetaVirus-GameData)) 使用**

1. 双击BattleTestUtility.exe启动测试工具
